E um codigo desenvolvido em python para gerenciamento de clientes em uma barbearia:
Cliente chega → O atendente cadastra ele no sistema, marcando quais serviços quer (corte, barba, etc.)
Cliente entra na fila → Ele aparece numa lista esperando sua vez
Barbeiro chama → Ao clicar num botão, o sistema "chama" o próximo cliente
Atendimento feito → O sistema registra o serviço e adiciona ao faturamento
